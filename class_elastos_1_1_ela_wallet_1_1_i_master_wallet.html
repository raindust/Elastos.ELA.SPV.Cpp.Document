Create a sub wallet by specifying wallet type.

Parameters
chainIDunique identity of a sub wallet. Chain id should not be empty.
payPassworduse to encrypt important things(such as private key) in memory. Pay password should between 8 and 128, otherwise will throw invalid argument exception.
singleAddressif true created wallet will have only one address inside, otherwise sub wallet will manager a chain of addresses for security.
feePerKbspecify fee per kb to calculate fee by size of transaction. Fee per key default set to zero so that sub wallet will calculate by default "fee rate".
Returns
If success will return a pointer of sub wallet interface.
