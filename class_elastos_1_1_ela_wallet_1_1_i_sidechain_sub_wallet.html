)
pure virtual

Create a withdraw transaction and return the content of transaction in json format. Note that amount should greater than sum of so that we will leave enough fee for mainchain.

Parameters
fromAddressspecify which address we want to spend, or just input empty string to let wallet choose UTXOs automatically.
toAddressspecify which address we want to send, in this method to address shall be destroy address of the side chain. Note that destroy address is a special address generate by a program hash that all bytes set to zero.
amountspecify amount we want to send.
mainchainAccountsa list of mainchain accounts in json format.
mainchainAmountsa list of mainchain amounts in json format, each amount should correspond to mainchainAccounts by order.
mainchainIndexsa list of mainchain indices in json format, each index should correspond to mainchainAccounts by order.
memoinput memo attribute for describing.
remarkis used to record message of local wallet.
Returns
If success return the content of transaction in json format.
